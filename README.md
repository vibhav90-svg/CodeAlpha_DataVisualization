# Real Estate Data Visualization

## Objective
To visualize real estate data and identify relationships between property features and prices.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Visualizations
- Property Type Distribution
- Furnishing Distribution
- Location Distribution
- Price Distribution
- Rooms vs Price
- Area vs Price
- Pool vs Price
- Correlation Heatmap

## Author
Vibhav Pandey# CodeAlpha_DataVisualization
Data Visualization project on a Real Estate Dataset using Python, Pandas, Matplotlib, and Seaborn. The project explores property trends, price distribution, location analysis, and relationships between property features through interactive visualizations.
